Ad-Types & Formats

Here are all different types of ads that we provide, choose the best one you need.

Banner Ads

It should be either 300x250 medium rectangle or 160x600 wide skyscraper or 728x90 leaderboard. The image type should be PNG or JPG.

Pop-Ups Ads

It should only be 300x250 rectangle, and a PNG or JPG image type. It will appear on the website when the user browses the website for one minute and the ad should be related to the topic.

Video Ads

Video dimensions should be 1280x720. Aspect ratio is 16:9 Landscape. Video type should be MP4 or MOV. Max file size is 1GB. Max length is two minutes.
